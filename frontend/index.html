About This System

This AI-powered DDoS Detection System uses XGBoost machine learning models trained on the CICDDoS2019 dataset to analyze network traffic flow data and detect DDoS attacks in real-time.

🎯 Capabilities

  • ✓ Detects 14 different DDoS attack types (SYN Flood, UDP Flood, DNS Amplification, etc.)
  • ✓ Predicts severity scores (0-100%) for each network flow
  • ✓ Identifies critical threats (>70% severity) for immediate response
  • ✓ Provides comprehensive attack distribution and severity statistics

📤 How to Use

  1. 1 Prepare a CSV file containing network traffic flow data with 81 specific features
  2. 2 Upload the CSV file using the drag-and-drop area or click to browse
  3. 3 View the analysis results including attack types, severity scores, and critical alerts

📊 Required Data Format

Your CSV file must contain exactly 81 columns in the correct order. Click the "Column Reference" button to see the full list.

⚠️ Important Notes

  • • File size limit: 100MB
  • • Processing time: 1-10 seconds depending on file size
  • • API rate limit: 10 uploads per minute
  • • Non-numeric columns (IP addresses, timestamps) are automatically removed

📋 Required CSV Columns

81 columns in exact order • CICDDoS2019 dataset format
